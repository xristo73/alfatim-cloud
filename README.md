# ☁ AlfaTim Cloud

Lekka, szybka i nowoczesna chmura plików napisana w **Python (Flask)**.

AlfaTim Cloud to projekt rozwijany z myślą o użytkownikach, którzy chcą mieć pełną kontrolę nad swoimi danymi bez konieczności korzystania z ciężkich rozwiązań typu Nextcloud. Głównym celem projektu jest prostota, szybkość działania oraz możliwość uruchomienia własnej chmury na dowolnym serwerze VPS lub komputerze domowym.

---

# ✨ Najważniejsze funkcje

* 📁 Zarządzanie plikami i folderami
* 📂 Przesyłanie całych folderów metodą Drag & Drop
* 📦 Automatyczne rozpakowywanie archiwów ZIP
* 🖼️ Miniatury obrazów
* 🔍 Wyszukiwarka plików i folderów
* 🗑️ Kosz z możliwością przywracania plików
* 🕓 Wersjonowanie plików
* 📋 Kopiowanie plików i folderów
* 📂 Przenoszenie plików i folderów
* ✏️ Zmiana nazw plików i folderów
* 📤 Udostępnianie plików za pomocą linków
* 👥 Obsługa wielu użytkowników
* 🔐 Logowanie oraz autoryzacja użytkowników
* 👨‍💼 Panel administratora
* 💾 Limity przestrzeni dyskowej dla użytkowników
* 📱 Responsywny interfejs działający na komputerach, tabletach i telefonach

---

# 🚀 Dlaczego powstał ten projekt?

Większość dostępnych rozwiązań do prywatniej chmury oferuje ogromną liczbę funkcji, których wielu użytkowników nigdy nie wykorzystuje. Powoduje to większe wymagania sprzętowe, wolniejsze działanie oraz bardziej skomplikowaną konfigurację.

AlfaTim Cloud powstała jako lekka alternatywa, skupiająca się na najważniejszych funkcjach potrzebnych do codziennego zarządzania plikami.

# 📂 Struktura projektu

```
app.py                 Główna aplikacja Flask
config.py              Konfiguracja projektu
database.py            Obsługa bazy danych

routes/                Trasy aplikacji
services/              Logika biznesowa
utils/                 Funkcje pomocnicze

templates/             Szablony HTML
static/                JavaScript, obrazy i pliki statyczne

uploads/               Pliki użytkowników
```
